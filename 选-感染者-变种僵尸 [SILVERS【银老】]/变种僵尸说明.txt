类型：

炸弹：【黄】
炸弹变种人在攻击、受伤（防御）或头部中弹时有随机几率爆炸。

火：【红】
在燃烧弹或烟花箱爆炸中行走的普通火有随机几率变异成火变种人。这些僵尸是防火的，在攻击或防御时有几率掉火。

幽灵：【白】
半透明僵尸。

心灵：【紫】
靠近这些僵尸会改变你的屏幕颜色。

烟雾：【青】
靠近这些僵尸的玩家会受到伤害。

唾液：【绿】
常见的穿过唾液酸的物种有随机几率变异成唾液突变体。当被这些僵尸击中时，玩家会受到几秒钟的伤害。

特斯拉：【蓝】
尼古拉·特斯拉变种人，将玩家从变种人手中甩开。


------------管理员命令-----------------
sm_mutantsrefresh   // 刷新插件并重新加载数据配置。
sm_mutantbomb       //生成一个变异炸弹僵尸。
sm_mutantfire       // 生成一个变异的火僵尸。
sm_mutantghost      //生成一个变异的幽灵僵尸。
sm_mutantmind       // 生成一个变异的心灵僵尸。
sm_mutantsmoke      // 生成一个变异烟雾僵尸。
sm_mutantspit       //生成一个变异的吐痰僵尸。
sm_mutanttesla      //生成一个变异的特斯拉僵尸。
sm_mutants          // 生成所有变异僵尸
