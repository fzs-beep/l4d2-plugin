玩家控制台输入命令使用


------普通玩家可输入指令------


sm_hat // 显示帽子菜单，允许玩家更改他们的穿着。可选参数：[0 - 128或帽子名称或“随机”]

sm_hats // 显示一个菜单来自定义帽子的各种设置。

sm_hatoff // 切换以打开或关闭戴帽子的能力。

sm_hatshow // 切换以查看或隐藏自己的帽子。使用可选命令参数“tp”适用于第一人称视图或第三人称，例如“sm_hatshow tp”

sm_hatview // ^^
sm_hatshowon // 看看你自己的帽子。使用可选命令参数“tp”应用于第一人称视图或第三人称，例如“sm_hatshowon tp”

sm_hatshowoff // 隐藏自己的帽子。使用可选命令参数“tp”适用于第一人称视图或第三人称，例如“sm_hatshowoff tp”

sm_hatall // 切换每个人帽子的可见性。

-------管理员权限操作-------

sm_hatclient // 设置客户端帽子。用法：sm_hatclient <#userid|name> [帽子名称或帽子索引：0-128（MAX_HATS）]。

sm_hatc // 显示一个菜单，列出播放器，选择一个来更改他们的帽子。

sm_hatallc // 切换特定玩家上所有帽子的可见性

sm_hatoffc // 切换特定玩家戴帽子的能力。

sm_hatrandom // 随机化所有玩家的帽子。

sm_hatrand // ^^

☆使用这些命令添加和删除帽子模型↓↓↓↓↓

sqm_hatadd // 将指定的模型添加到配置中（必须是完整的模型路径）。

sm_hatdel // 从配置中删除模型（通过索引或部分名称匹配）。

sm_hatlist // 显示所有帽子模型和索引的列表（与sm_hatdel一起使用）。

☆这些命令编辑帽子外观↓↓↓↓↓↓↓

sm_hatsave // 保存帽子原点和角度到数据配置。

sm_hatload // 将所有玩家的帽子更改为您拥有的帽子。

sm_hatpos // 显示一个菜单，允许您调整帽子位置（影响所有帽子/玩家）。

sm_hatang // 显示一个菜单，允许您调整帽子角度（影响所有帽子/玩家）。

sm_hatsize // 显示一个菜单，允许您调整帽子尺寸（影响所有帽子/玩家）。
