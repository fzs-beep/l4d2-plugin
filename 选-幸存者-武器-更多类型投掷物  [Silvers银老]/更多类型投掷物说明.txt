翻译：
20种不同的类型
自定义 \data\l4d_grenades.cfg 文件中的许多选项。
自定义允许的类型以及它们适用于哪种手榴弹。
更改手榴弹类型的菜单。可与换键方法组合使用。
玩家可以装备手榴弹并按住 SHOOT （LMB） + SPUSH （RMB） 键来更改手榴弹类型。
机器人可以使用原型手榴弹。需要外部插件。
拾取手榴弹时的随机手榴弹类型或客户偏好保存每个手榴弹的选定类型。
添加新的手榴弹类型应该很容易，所以请随时提出想法。


原文：
20 different types.
Customize lots of options in the \data\l4d_grenades.cfg file.
Customize which types to allow and which grenade they work for.
Menu to change the grenade type. Can be used in combination with the key changing method.
Players can equip a grenade and hold SHOOT (LMB) + SHOVE (RMB) keys to change grenade type.
Bots can use Prototype Grenades. Requires external plugin.
Random grenade type when picking up or client preferences to save selected type for each grenade.
Adding new grenade types should be easy so feel free to suggest ideas



----------炸弹类型----------------
燃烧弹：
- 炸弹：爆炸性手榴弹。撞击时引爆。
- 集群：射出多个爆炸性射弹。
- 烟花：烟花箱爆炸。  
- 烟雾：烟幕隐藏区域。
- 集群火力：发射多个小火。
- 高射炮：发射火花点燃附近的敌人。
- 空袭：火炬发出空袭目的地的信号。（需要按装“F-18 空袭”插件）

土质炸弹：
- 黑洞：将敌人拉向中心。
- 闪光弹：引爆时使幸存者和特殊感染者失明。
- 护盾：减少对区域内幸存者的伤害。
- 特斯拉：电击区域内的敌人，将他们击退。
- 子弹：射出多颗子弹。
- 发光：在短时间内透过墙壁发光。
- 武器：在爆炸时制造武器。通过插件配置选择类型。

胆汁：（在L4D1[求生之路1代游戏]中这些类型分为燃烧弹和土质炸弹）
- 化学：地面上的酸性水坑以伤害敌人。
- 冰冻器：冻结影响范围内的敌人。
- 医生：治疗 - 仅对幸存者有效。
- 蒸发器：溶解区域内的敌人。
- 灭火器：扑灭地面和玩家的火势，可以消除婴儿潮呕吐效果和喷出酸（L4D2）。
- 反重力：使敌人跳起来并慢慢落下。
